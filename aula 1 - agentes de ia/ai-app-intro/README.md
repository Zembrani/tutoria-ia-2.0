# Como rodar a aplicação com Poetry

```bash
pip install poetry
```

1. Instale as dependências:
```bash
poetry install
```

2. Execute a aplicação:
```bash
poetry run python src/main.py
```

```bash
poetry env info
```

Se necessário, ative o ambiente virtual do Poetry:
```bash
poetry lock
```
